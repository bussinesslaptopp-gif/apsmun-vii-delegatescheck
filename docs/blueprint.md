# **App Name**: Delegate Directory

## Core Features:

- CSV Parsing and Loading: Dynamically parse the delegate data from a CSV file, handling potential errors and extra columns.
- Delegate Information Display: Display the delegate's name, DelegateNo, and committee in a clear and organized format.
- QR Code Scanner: Integrate a QR code scanner that reads DelegateNo from the QR code and retrieves the associated delegate information.
- Manual DelegateNo Input: Allow users to manually enter a DelegateNo to search for delegate information.
- Search by Name or DelegateNo: Implement a search feature to find delegates by name or DelegateNo.
- Delegate Not Found Handling: Display a 'Delegate not found' message if the entered DelegateNo or name does not exist in the CSV data.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust. The choice is inspired by the standard colors of formal events.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) for interactive elements and highlights, creating contrast without being jarring.
- Body and headline font: 'PT Sans', a modern humanist sans-serif font.
- Use simple and professional icons for search and QR code scanning functions.
- Mobile-first, responsive layout for use across devices. Keep it streamlined, with the search or scanning feature easily available upon opening the site.
- Subtle transitions on information display.
